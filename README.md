# 🎓 LearnSphere

A responsive online learning platform that allows users to browse courses, enroll, take quizzes, and track their learning progress.

Built using HTML, CSS, JavaScript, Bootstrap, and Angular.

---

## 🚀 Live Demo
https://aliyarashidsarhan-create.github.io/LearnSphere/

---

## 📁 Project Structure

- **docs/** → HTML, CSS, JavaScript version (deployed on GitHub Pages)  
- **angular-app/** → Angular SPA version (presented locally)

---

## ✨ Key Features

- Dynamic course catalog from JSON data  
- Search, filter, and sorting system  
- Course enrollment using **localStorage**  
- Quiz system with score calculation  
- Learning dashboard with progress tracking  
- Dark mode with saved user preference  
- Angular SPA with components, routing, and services  

---

## 🛠️ Technologies Used

- HTML5 & CSS3  
- Bootstrap 5  
- JavaScript (ES6+)  
- Angular 16  
- TypeScript  
- LocalStorage
